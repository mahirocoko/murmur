# Mahiro Whisper

Personal macOS dictation app built with Tauri v2 and whisper.cpp.

- Product/architecture plan: [`docs/plan.md`](docs/plan.md)
- Local development notes: [`docs/development.md`](docs/development.md)

## Quick Start

```sh
pnpm install
pnpm build
cd src-tauri && cargo check
pnpm tauri dev
```
